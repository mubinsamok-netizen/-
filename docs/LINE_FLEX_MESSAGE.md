# LINE Flex Message ที่ระบบส่ง

ระบบส่งข้อความ 2 ส่วน:

1. ข้อความสุภาพแบบ text
2. Flex card หัวข้อ `แจ้งใบวางบิล`

ปุ่มใน Flex card:

```text
เปิดเอกสาร
แจ้งชำระเงิน
```

ตัวอย่างประโยคแจ้งวางบิล:

```text
เรียนคุณลูกค้า

ขออนุญาตส่งใบวางบิลประจำรอบนี้ให้ตรวจสอบค่ะ/ครับ
ยอดชำระ {amount} บาท
ครบกำหนดชำระวันที่ {due_date}

สามารถเปิดดูรายละเอียดได้จากปุ่มด้านล่าง
หากชำระเรียบร้อยแล้ว สามารถแจ้งหลักฐานการชำระเงินกลับทาง LINE นี้ได้เลยค่ะ/ครับ

ขอบคุณมากค่ะ/ครับ
ฝ่ายบัญชี
```

ข้อความเตือนก่อนครบกำหนดและติดตามหลังเลยกำหนดอยู่ในไฟล์:

```text
netlify/functions/_shared/line.ts
```
